Olá! para o uso da planilha, siga as instruções abaixo.

1. Escolha a pasta 'Raiz' utilizando o botão 'Configurações > Trilha de dados'.
2. Na pasta raiz será criada uma pasta 'lattes' que conterá os arquivos.
3. Clique em 'Configurações > Inserir > Nova estrutura' para criar a estrutura do currículo.
4. Para visualizar o caminho das pastas, use o botão 'Configurações > Pastas'.
5. Para cadastrar uma nova pasta, clique em 'Configurações > Inserir > Nova pasta'.
6. Utilize os botões na guia 'Cadastros' para cadastrar novas informações.
7. Ao cadastrar uma informação será necessário selecionar um arquivo, que será automaticamente identificado e copiado para a pasta específica.
8. Para remover um registro, utilize o botão 'remove' disponível em cada planilha. O registro e o arquivo correspondente serão deletados.

Bom uso!
